## 특별시 포지셔닝 전략

### 포지셔닝 맵

```
              고가/프리미엄
                   ↑
                   |
    숨고(일부)     |     특별시
    크몽(일부)     |     (웨딩 특화)
                   |
←─ 범용 ──────────┼────────────→ 버티컬
                   |
    숨고(대부분)   |     웨딩박람회
    크몽(대부분)   |     (오프라인)
                   |
                   ↓
              저가/대중
```

### 특별시 포지셔닝

> **"웨딩 서브서비스 전문 AI 매칭 플랫폼"**
> - 버티컬 집중: 웨딩 서브서비스만
> - AI 차별화: 스타일 기반 매칭
> - 프리미엄 포지션: 품질 우선

## AI 매칭 로직 상세

### 매칭 알고리즘 구조

```
1. 예비부부 입력
   ├── 스타일 선호 (모던/클래식/빈티지 등)
   ├── 예산 범위
   ├── 결혼식 날짜/장소
   └── 참고 이미지 업로드 (선택)

2. AI 분석
   ├── 자연어 처리 (스타일 키워드 추출)
   ├── 이미지 임베딩 (참고 이미지 분석)
   └── 협업 필터링 (유사 고객 선호 반영)

3. 프리랜서 매칭
   ├── 포트폴리오 스타일 벡터
   ├── 가격/일정 필터
   └── 리뷰 점수 가중치

4. 추천 결과
   └── Top 5 프리랜서 + 매칭 점수
```

### 기술 스택

- **자연어 처리**: OpenAI GPT / Claude API
- **이미지 분석**: CLIP / ResNet 기반 임베딩
- **추천 엔진**: Matrix Factorization + Neural CF
- **백엔드**: FastAPI + PostgreSQL
- **프론트엔드**: Flutter (모바일), Next.js (웹)

## 웨딩 크루 시스템 (차별화 기능)

### 개념

> **"원클릭 패키지 매칭"**
> 스냅+영상+축가+사회자 등을 팀으로 묶어 한 번에 매칭

### 작동 방식

```
1. 예비부부: "본식 풀패키지 원해요"
   └── AI가 스타일에 맞는 크루 자동 구성

2. 웨딩 크루 제안
   ├── 스냅 작가 A (스타일 매칭 95%)
   ├── 영상 작가 B (스타일 매칭 92%)
   ├── 축가 가수 C (스타일 매칭 88%)
   └── 총 패키지 가격: 120만원

3. 예비부부: 크루 확정 또는 개별 수정

4. 크루 내 협업
   └── 공유 일정표, 촬영 동선 조율, 결과물 통합 전달
```

### 크루 혜택

| 대상 | 혜택 |
|------|------|
| **예비부부** | 개별 섭외 시간 80% 절감, 통일된 스타일, 패키지 할인 |
| **프리랜서** | 교차 판매 기회, 협업 네트워킹, 안정적 수입 |
| **특별시** | 객단가 3배 증가, 프리랜서 Lock-in |

## 커뮤니티 전략 (유니콘 성공 공식)

### 핵심 원리

> **"커뮤니티가 플랫폼을 만든다"**
> 에어비앤비, 크래프톤, 무신사 모두 커뮤니티에서 시작

### 웨딩 라운지 (커뮤니티)

```
웨딩 라운지
├── 예비부부 채널
│   ├── 웨딩 준비 Q&A
│   ├── 후기 공유 (작가 태그)
│   └── 스타일 투표
│
└── 프리랜서 채널
    ├── 포트폴리오 공유
    ├── 노하우 공유
    └── 협업 구인
```

### 3C 플라이휠

```
Content (콘텐츠)
   ↓
Community (커뮤니티)
   ↓
Commerce (거래)
   ↓
   (반복)
```

1. **Content**: 후기/포트폴리오로 유입
2. **Community**: 질문/답변으로 참여
3. **Commerce**: 매칭 거래로 수익화
4. 거래 후 후기 → 다시 Content

## 글로벌 유사 서비스 벤치마크

### Thumbtack (미국)

| 항목 | 내용 |
|------|------|
| **분야** | 홈서비스 매칭 |
| **기업가치** | $3.5B |
| **핵심 전략** | AI 자동 매칭, 지역 기반 밀도 |
| **벤치마크** | 버티컬 집중 → 지역 확장 |

### Thumbtack 성공 공식

1. **버티컬 집중**: 홈서비스만
2. **AI 매칭**: 자동 프로 추천
3. **지역 밀도**: 도시별 순차 런칭
4. **공급자 우선**: 프로 확보 → 수요 폭발

### 특별시 적용

- Thumbtack 전략을 웨딩 버티컬에 적용
- 서울/경기 → 지방 순차 확장
- AI 스타일 매칭으로 차별화
