# START. 통합 준비 일정표 (26년)

> 뉴콘텐츠아카데미(26년 1~8월) + 26년 예창패를 동시에 진행하는 실전 로드맵

---

## 핵심 전략

뉴콘텐츠아카데미(26년 1~8월)와 예창패(26년 2~12월)를 **병행 진행**합니다.
아카데미에서 만드는 MVP와 검증 데이터가 곧 예창패 신청 자료가 됩니다.

---

## 총 지원금 1.2억원 구조

### 뉴콘텐츠아카데미: 2,000만원
- **기간**: 26년 1~8월 (8개월)
- **제약**: 인건비 사용 불가
- **가능**: 개발비, 마케팅비, 서버, 운영비

### 예비창업패키지: 1억원
- **기간**: 26년 6~12월 (7개월)
- **인건비 포함 가능**
- **가능**: 인건비, 개발비, 마케팅비, 모든 비용

## 월별 핵심 준비사항 체크리스트

### Phase 1: 기획 + 신청 (26년 1~3월)

**목표: 예창패 신청 완료 + 아카데미 기획 완료**

**예창패 신청 (3월 마감)**
- [ ] 사업계획서 최종 완성
- [ ] K-Startup 온라인 접수
- [ ] 발표 피칭덱 초안 준비
- [ ] 예상 질문 30개 준비

**아카데미 기획**
- [ ] UX/UI 와이어프레임 완성
- [ ] 심층인터뷰 20명+ 진행
- [ ] 페인포인트 분석 보고서
- [ ] 기술 스택 확정

### Phase 2: 개발 + 평가 (26년 4~5월)

**목표: MVP 완성 + 발표평가 통과**

**예창패 평가 대응 (5월)**

- [ ] 모의 발표 10회 이상
- [ ] MVP 시연 준비 (있으면 가산점!)
- [ ] 아카데미 검증 데이터로 답변

**아카데미 개발**
- [ ] 핵심 기능 MVP 완성
- [ ] 설문조사 100명+ 완료
- [ ] PMF 점수 측정
- [ ] 내부 알파 테스트

### Phase 3: 런칭 + 사업수행 (26년 6~8월)

**목표: 베타 런칭 + 예창패 선정 후 자금 집행**
