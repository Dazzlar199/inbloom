#### 1. 문제인식 (25점)
- [ ] 고객 페인포인트 3개 이상 구체적 기술

- [ ] 경쟁사 비교표 포함

#### 3. 성장전략 (25점)
- [ ] Unit Economics (LTV/CAC) 산출

- [ ] 3년 재무계획 테이블
- [ ] 콜드스타트 해결 전략
- [ ] 시장 확장 로드맵

#### 4. 팀구성 (25점)

- [ ] 팀 스토리 (Why Us?)

## 발표심사 체크리스트

### 발표 구성 (7분)

| 시간 | 내용 | 비중 |
|------|------|------|
| 0:00-0:30 | 후킹 (문제 제기) | 7% |
| 0:30-1:30 | 문제인식 | 15% |
| 1:30-3:00 | 해결방안 (데모 포함) | 21% |
| 3:00-4:30 | 시장/경쟁분석 | 21% |
| 4:30-5:30 | 사업모델/재무 | 15% |
| 5:30-6:30 | 팀 소개 | 15% |
| 6:30-7:00 | 클로징 | 7% |

### 발표 준비

- [ ] 7분 스크립트 완성 (외우기 X, 핵심만)
- [ ] PPT 20장 내외 (1장당 20초)
- [ ] 데모 영상 준비 (30초 내외)
- [ ] 백업 PPT 준비 (USB + 클라우드)

### Q&A 준비 (8분)

#### 반드시 나오는 질문

1. **"숨고/크몽과 뭐가 달라요?"**
   - 답변 준비: 웨딩 특화, AI 매칭, 고정 수수료

2. **"콜드스타트는 어떻게?"**
   - 답변 준비: CEO 네트워크, 인센티브 설계

3. **"수익성 있나요?"**
   - 답변 준비: LTV/CAC 8배, Unit Economics

4. **"팀이 왜 이걸 해야 하나요?"**
   - 답변 준비: 창업 스토리, 도메인 경험

5. **"이미 30명 인터뷰했다는데, 핵심 인사이트는?"**
   - 답변 준비: Top 3 페인포인트 + 수치

#### 까다로운 질문 예상

6. **"플랫폼 우회 거래는 어떻게 막나요?"**
7. **"숨고가 웨딩 카테고리 강화하면?"**
8. **"AI 매칭 정확도 어떻게 검증했나요?"**
9. **"3년 후 프리랜서 1만명 근거는?"**
10. **"투자 유치 실패하면 어떻게 할 건가요?"**

### 발표 당일 체크리스트

**전날**
- [ ] PPT 최종 점검
- [ ] 노트북 충전 100%
- [ ] 어댑터/케이블 준비
- [ ] 복장 준비 (비즈니스 캐주얼)

**당일 아침**
- [ ] 발표장 위치 재확인
- [ ] 1시간 전 도착 목표
- [ ] 물 준비
- [ ] 스트레칭 (긴장 완화)

**발표 직전**
- [ ] PPT 테스트 실행
- [ ] 마이크 테스트
- [ ] 시간 체크 (타이머 사용)
- [ ] 심호흡 3회

## 핵심 요약: 합격을 위한 5가지 핵심

### 1. 실제 검증 데이터
> 인터뷰 35명, 설문 150명 - 숫자로 증명

### 2. 차별화된 솔루션
> AI 스타일 매칭 + 고정 수수료 - 기존 플랫폼과 명확히 다름

### 3. 탄탄한 사업성
> LTV/CAC 8배, 11개월 BEP - 투자자 기준 충족

### 4. 실행력 있는 팀
> 도메인 경험 + 기술 역량 + 이미 실행 중

### 5. 명확한 성장 경로
> 웨딩 → 경조사 → 기업행사 (TAM 15조원)
