## BEP 분석

### 월간 고정비

| 항목 | 금액 | 비고 |
|------|------|------|
| 인건비 | 800만원 | 3인 기준 |
| 서버/인프라 | 50만원 | AWS 기준 |
| 마케팅 | 350만원 | 초기 집중 |
| 기타 운영비 | 100만원 | 사무실, 툴 등 |
| **월 고정비 합계** | **1,300만원** | |

### 월간 변동비

| 항목 | 비율 |
|------|------|
| PG 수수료 | 거래액의 2.5% |
| CS 비용 | 매출의 5% |

### BEP 산출

#### 구독 수익 기준
```
BEP 구독자 수 = 월 고정비 / 평균 구독료
             = 1,300만원 / 19,900원
             = 653명
```

#### 거래 수수료 포함 시
```
월 고정비: 1,300만원
구독 수익 (400명 × 15,000원): 600만원
필요 거래 수수료: 700만원
필요 거래 건수: 700만원 / 4,000원 = 1,750건
```

### BEP 달성 시점 예측

| 시점 | 프리랜서 수 | 구독자 수 | 월 거래 건수 | BEP 달성 |
|------|-----------|----------|------------|---------|
| 6개월차 | 300명 | 200명 | 500건 | 50% |
| 9개월차 | 600명 | 400명 | 1,200건 | 85% |
| **11개월차** | **800명** | **550명** | **1,750건** | **100%** |
| 12개월차 | 1,000명 | 700명 | 2,200건 | 120% |

> **결론**: 11개월차에 BEP 달성 예상
