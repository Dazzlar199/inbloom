## [요약] 심사위원용 1페이지 요약

### 한 줄 소개
**특별시**는 웨딩 서브서비스(스냅, 영상, 축가 등) 프리랜서와 예비부부를 AI 스타일 매칭으로 연결하고, 건당 고정 수수료(최대 5,000원)로 수익을 창출하는 버티컬 플랫폼이다.

### 핵심 고객 및 문제

| 고객 | 핵심 문제 |
|------|----------|
| **예비부부** | 작가 스타일 비교 어려움, 가격 불투명, 피해 사례 우려 |
| **프리랜서** | 숨고/크몽 10~20% 높은 수수료, 가격 경쟁 압박, 포트폴리오 노출 한계 |

### 해결방안 요약
- **AI 스타일 매칭**: 포트폴리오 이미지 분석 기반 자동 추천 (3주 → 3분)
- **고정 수수료**: 건당 5,000원 상한 (기존 대비 70% 비용 절감)
- **에스크로 결제**: 서비스 완료 후 정산으로 피해 방지

### 시장규모 및 수익모델
- TAM: 2조원 (연 19만 쌍 × 평균 100만원 서브서비스 지출)
- 수익: 매칭 수수료 + 정산 수수료 + 프리랜서 구독료

### 예창패 기간 내 핵심 목표 (8개월)

| 지표 | 목표 |
|------|------|
| 프리랜서 확보 | 100~150명 |

## 1. 창업아이템 개요 및 필요성

### 1.1 사업 개요

**특별시**는 웨딩 서브서비스(스냅촬영, 영상, 축가, 사회자 등) 분야에 특화된 AI 기반 프리랜서 매칭 플랫폼이다.

- **고객**: 결혼을 준비하는 예비부부 (연 19만 쌍)
- **공급자**: 웨딩 관련 프리랜서 (스냅작가, 영상감독, 축가가수 등)
- **핵심 가치**: AI 스타일 매칭으로 탐색 시간 단축, 고정 수수료로 프리랜서 실수령액 극대화

### 1.2 해결하고자 하는 문제

#### 예비부부 (수요자) 페인포인트

| 문제 | 현황 | 영향 |
|------|------|------|
| 작가 탐색 어려움 | 숨고/크몽에서 평균 3주 소요 | 결혼 준비 스트레스 가중 |
| 가격 불투명 | 동일 서비스 2~3배 가격 차이 | 과다 지출 또는 피해 |
| 품질 불확실 | 리뷰 조작, 포트폴리오 과장 | 결혼식 당일 실망 |

*"숨고에서 축가 작가를 찾는 데만 3주가 걸렸고, 가격 비교가 안 돼서 더 비싸게 계약한 적이 있다."* - 인터뷰 참여자 A (30대, 2024년 결혼)

#### 프리랜서 (공급자) 페인포인트

| 문제 | 현황 | 영향 |
|------|------|------|
| 높은 수수료 | 숨고 10~20%, 크몽 20% | 실수령액 감소 |
| 가격 경쟁 | 최저가 경쟁 압박 | 품질 저하 악순환 |
| 고객 확보 한계 | 인스타그램 의존 | 신규 진입 장벽 |

*"수수료가 20%나 되니까 결국 가격을 올릴 수밖에 없고, 그러면 고객이 줄어든다."* - 인터뷰 참여자 B (웨딩스냅 5년차)

### 1.3 시장 규모

#### 웨딩 시장 구조

| 구분 | 규모 | 비고 |
|------|------|------|
| 웨딩 시장 전체 | 15조원 | |
| 스드메 (스튜디오/드레스/메이크업) | 5조원 | |
| 예식장 | 4조원 | |
| 혼수/예물 | 4조원 | |
| **서브서비스** | **2조원** | **특별시 타겟** |

#### TAM/SAM/SOM

| 구분 | 규모 | 산출 근거 |
|------|------|----------|
| TAM | 2조원 | 연 19만 쌍 × 평균 100만원 |
| SAM | 4,000억원 | 온라인 플랫폼 이용 40% × 수도권 50% |
| SOM (3년차) | 40~80억원 | 목표 점유율 1~2% |

### 1.4 왜 지금인가

1. **MZ세대 결혼 문화 변화**: 스몰웨딩 증가, 개별 커스터마이징 선호
2. **SNS 영향력 확대**: 웨딩스냅 인스타 게시물 120만+, 숏폼 콘텐츠 영향력 증가
3. **AI 기술 성숙**: 이미지 분석/추천 기술 상용화 가능 수준 도달
4. **국가적 과제**: 저출산 문제 해결을 위한 결혼 비용 부담 완화 필요성

## 2. 문제 인식 및 해결 방안

### 2.1 현재 시장의 비효율

| 구분 | 숨고/크몽 (범용 플랫폼) | 특별시 (버티컬 플랫폼) |
|------|------------------------|----------------------|
| 포지션 | 전 서비스 영역 | 웨딩 서브서비스 특화 |
| 매칭 방식 | 키워드 검색 | AI 스타일 매칭 |
| 수수료 | 10~20% | 고정 5,000원 상한 |
| 결제 | 외부/직거래 | 에스크로 결제 |
| 웨딩 특화 기능 | 없음 | 웨딩 크루, 일정 관리 |

### 2.2 특별시 서비스 개요

#### 핵심 기능 3가지

**1. AI 스타일 매칭**

| 단계 | 내용 |
|------|------|
| 입력 | 예비부부가 선호 스타일 선택 + 참고 이미지 업로드 |
| 처리 | 이미지 임베딩 분석 → 포트폴리오 스타일 벡터 매칭 |
| 출력 | 매칭 점수 상위 5명 프리랜서 추천 |

- 기대 효과: 탐색 시간 3주 → 3분 (99% 단축)
- 기술: CLIP 기반 이미지 임베딩, 협업 필터링

**2. 고정 수수료 체계**

| 거래액 구간 | 정산 수수료 | 숨고 대비 절감 |
|------------|-----------|--------------|
| 10만원 미만 | 1,000원 | 90% |
| 10~30만원 | 2,000원 | 85% |
| 30~50만원 | 3,000원 | 80% |
| 50만원 이상 | 4,000원 (상한) | 75% |

*매칭 수수료 1,000원 별도, 총 최대 5,000원*

- 고객 가치: 프리랜서 실수령액 증가 → 품질 향상 → 고객 만족도 증가

**3. 에스크로 안전 결제**

- 결제 대금 플랫폼 임시 보관
- 서비스 완료 확인 후 프리랜서에게 정산
- 분쟁 발생 시 중재 절차 제공

### 2.3 차별화 요소 비교

| 차별점 | 기능 | 고객 가치 | 경쟁사 대비 |
|--------|------|----------|------------|
| AI 매칭 | 스타일 기반 자동 추천 | 탐색 시간 99% 단축 | 숨고/크몽에 없음 |
| 고정 수수료 | 5,000원 상한 | 비용 70% 절감 | 10~20% → 1% 미만 |
| 웨딩 크루 | 스냅+영상+축가 패키지 | 원클릭 매칭 | 개별 섭외 필요 없음 |

### 2.4 고객 가치 정량화

| 항목 | 기존 (숨고/크몽) | 특별시 | 개선 |
|------|-----------------|--------|------|
| 작가 탐색 시간 | 평균 21일 | 3분 | 99% 단축 |
| 수수료 부담 (40만원 거래) | 4~8만원 | 5,000원 | 93% 절감 |
| 피해 발생률 | 확인 불가 | 에스크로로 원천 차단 | - |
