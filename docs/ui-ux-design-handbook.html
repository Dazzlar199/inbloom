Chapter 00

디자인 시스템 개요

특별시 (The Special Time) 디자인 철학 및 원칙

0.1. 디자인 컨셉

특별시는 웨딩 프리랜서 매칭 플랫폼으로, 따뜻하고 로맨틱한 감성을 추구합니다.

핵심 가치

  • Warmth: 특별한 순간의 따뜻함
  • Romance: 로맨틱하고 우아한 분위기
  • Luxury: 프리미엄 웨딩 서비스
  • Trust: 신뢰감 있는 서비스

0.2. 디자인 원칙

원칙 설명 적용 예시
Clarity 모든 요소가 명확한 목적 불필요한 장식 제거, 명확한 계층 구조
Consistency 일관된 패턴 사용 동일한 버튼 스타일, 스페이싱 규칙
Accessibility 모든 사용자 접근 가능 WCAG AA 준수, 충분한 컬러 대비
Performance 빠르고 부드러운 경험 이미지 최적화, 애니메이션 최소화

0.3. 플랫폼별 적용

플랫폼별 차이점
  • Flutter (앱): iOS/Android 네이티브 가이드라인 준수, 터치 중심
  • Next.js (웹): SEO 최적화, 데스크탑/태블릿 고려, 마우스/키보드 지원
  • 공통: 컬러 팔레트, 타이포그래피, 컴포넌트 스타일 동일

2.0. 폰트 미리보기 도구

폰트를 선택하고 실시간으로 미리보기를 확인할 수 있습니다.

특별한 순간을 만들어드립니다
생성된 코드
CSS
font-family: 'Pretendard Variable', 'Pretendard', -apple-system, sans-serif; font-size: 34px; font-weight: 700;
Flutter
TextStyle(fontFamily: 'Pretendard Variable', fontSize: 34, fontWeight: FontWeight.w700)
사용 방법
  1. 폰트를 선택하세요 (Display 또는 Body)
  2. 미리보기 텍스트를 입력하거나 수정하세요
  3. 크기와 Weight를 조절하여 원하는 스타일을 찾으세요
  4. 생성된 CSS/Flutter 코드를 복사하여 사용하세요

2.1. 폰트 시스템

웨딩 플랫폼의 특성에 맞춰 Display 폰트와 Body 폰트를 분리하여 사용합니다.

폰트 조합 전략

용도 주 폰트 대체 폰트 사용 위치
Display
(고급스러운 제목)		 Cormorant Garamond
(Serif)		 Georgia, serif 히어로 타이틀, 큰 제목, 브랜드명
Body
(가독성 본문)		 Pretendard Variable
(Sans-serif)		 Noto Sans KR, -apple-system 본문, 버튼, 일반 텍스트
폰트 선택 원칙
  • 최대 2개 폰트 패밀리: Display 1개 + Body 1개
  • 무게 제한: 각 폰트당 최대 3가지 weight 사용 (300, 400/500, 600/700)
  • 로딩 성능: 필요한 weight만 로드

현재 프로젝트 폰트 설정

플랫폼 Display 폰트 Body 폰트
Flutter 앱 Pretendard Variable
(모든 용도 통합)		 Pretendard Variable
Next.js 웹 Cormorant Garamond
(웹 디자인 프로토타입)		 Noto Sans KR

폰트 설치 (Flutter)

# pubspec.yaml
flutter:
  fonts:
    # Display 폰트 (선택)
    - family: Cormorant Garamond
      fonts:
        - asset: assets/fonts/CormorantGaramond-Regular.ttf
          weight: 400
        - asset: assets/fonts/CormorantGaramond-SemiBold.ttf
          weight: 600
        - asset: assets/fonts/CormorantGaramond-Bold.ttf
          weight: 700
    
    # Body 폰트
    - family: Pretendard Variable
      fonts:
        - asset: assets/fonts/PretendardVariable.ttf

폰트 설치 (Next.js)

// app/layout.tsx
import { Cormorant_Garamond } from 'next/font/google'
import localFont from 'next/font/local'

// Display 폰트 (Google Fonts)
const cormorant = Cormorant_Garamond({
  weight: ['400', '600', '700'],
  subsets: ['latin'],
  variable: '--font-display',
  display: 'swap',
})

// Body 폰트 (로컬)
const pretendard = localFont({
  src: './fonts/PretendardVariable.woff2',
  variable: '--font-body',
  display: 'swap',
})

export default function RootLayout({ children }) {
  return (
    
      
        {children}
      
    
  )
}

CSS 변수 사용

/* globals.css */
:root {
  --font-display: 'Cormorant Garamond', Georgia, serif;
  --font-body: 'Pretendard Variable', 'Noto Sans KR', -apple-system, sans-serif;
}

/* Display 폰트 사용 */
.display-title {
  font-family: var(--font-display);
  font-weight: 700;
  font-size: 48px;
}

/* Body 폰트 사용 */
body {
  font-family: var(--font-body);
}

2.1-1. 대안 폰트 옵션

프로젝트 성격에 따라 다른 폰트 조합을 선택할 수 있습니다.

스타일 Display 폰트 Body 폰트 특징
럭셔리 웨딩

  • 폴백 전략: 시스템 폰트로 graceful degradation
  • 라이선스: 상업적 사용 가능 여부 확인

이름 값 Flutter CSS/Tailwind 용도

크기 값 용도

비율 값 용도
